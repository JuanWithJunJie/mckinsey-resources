# 麦肯锡人物时间线

## 历史发展

| 时间 | 事件 | 对思维的影响 |
|------|------|-------------|
| 1926 | James O. McKinsey在芝加哥创立 | 会计学背景奠定严谨方法论传统 |
| 1930s | Marvin Bower加入，后成为麦肯锡方法论奠基人 | 提出「以事实为基础，以假设为驱动，以逻辑为纽带」 |
| 1940s | Marvin Bower成为CEO，将麦肯锡从会计事务所转型为管理咨询 | 奠定现代咨询方法论基础 |
| 1960s | 麦肯锡方法论体系化 | 金字塔原理、MECE开始成型并广泛传播 |
| 1970s | 全球扩张，日本、韩国办公室成立 | 框架思考开始向亚洲传播 |
| 1980s | 《麦肯锡方法》前身思想形成 | 假设驱动、结构化分析成为核心方法 |
| 1990s | 芭芭拉·明托《金字塔原理》出版 | 金字塔原理成为咨询行业标准 |
| 2000s | 麦肯锡学院丛书系统性出版 | 方法论开始向大众普及 |
| 2010s | 数字化转型浪潮 | 麦肯锡方法论与新技术结合 |
| 2020s | AI时代 | 《Rewired》等新著作强调技术与AI驱动 |

## 关键人物

### James O. McKinsey (创始人)
- 芝加哥大学会计学教授
- 创立了将会计严谨性引入咨询的方法论

### Marvin Bower (方法论奠基人)
- 1939年加入麦肯锡
- 奠定了麦肯锡的文化和方法论基础
- 强调以事实为基础、以客户为中心

### 芭芭拉·明托 (金字塔原理创始人)
- 麦肯锡第一位女性咨询顾问
- 发明金字塔原理
- 1973年出版《金字塔原理》

### 当代传承者
- 赤羽雄二：《零秒思考》《麦肯锡精英高效阅读法》
- 大岛祥誉：《麦肯锡工作法》《麦肯锡思考工具》
- 高杉尚孝：《麦肯锡问题分析与解决技巧》《麦肯锡教我的写作武器》
