## 使用方法 / Usage

激活 skill 后，用中文或英文描述你的问题或挑战，系统会调用麦肯锡的7个核心思维框架帮你分析。

After activating the skill, describe your problem or challenge in Chinese or English, and the system will apply McKinsey's 7 core thinking frameworks to help analyze it.
